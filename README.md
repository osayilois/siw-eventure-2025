## 🌐 Overview

Eventure is a full-stack web application designed to manage and discover local events.

The platform allows users to browse events, while organizers can create and manage them through a role-based system. The application is built using Spring Boot and follows a structured backend architecture with authentication and access control.

## ✨ Features

User authentication and registration

Event creation and management

Role-based access control (USER, ORGANIZER, ADMIN)

Event browsing and listing

Secure interaction between users and organizers

## 🛠 Tech Stack

Java

Spring Boot

HTML / CSS

Git

## 🏗 Architecture

The application follows a typical Spring Boot MVC architecture, structured into:

Controllers → manage HTTP requests

Services → handle business logic

Repositories → manage database access

Domain models → represent core entities (User, Event, etc.)

Role-based authorization ensures that users interact with the system according to their permissions.
